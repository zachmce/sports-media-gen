# Security Policy

matchup-thumbs is a personal/homelab project. It is maintained best-effort,
by one person, alongside other work — please read the expectations below
before reporting.

## Reporting a Vulnerability

**Do not open a public issue for a security vulnerability.**

Please report vulnerabilities privately through GitHub's private vulnerability
reporting flow:

1. Go to the [Security tab](../../security) of this repository.
2. Click **Report a vulnerability**.
3. Describe the issue, including steps to reproduce and potential impact.

This routes the report directly to the maintainer and keeps details private
until a fix is available.

## Supported Versions

Only the latest released version is supported. There is no maintenance
branch, and older tags do not receive backported fixes.

| Version | Supported |
| ------- | --------- |
| latest (v3.0.1) | Yes |
| < v3.0.1 | No |

## Response Expectations

This is a personal/homelab project with no formal SLA. Reports are reviewed
and addressed on a best-effort basis as time allows — there is no guaranteed
response window. Thank you for reporting responsibly.
